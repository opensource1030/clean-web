## Installation

### Install dependencies

``` bash
# install app's dependencies
npm install

```

## Usage

``` bash
# serve with hot reload at localhost:8080
# you should check the all environment variables in `config/dev.env.js`
npm run start

# build for production with minification
# you should check the all environment variables in `config/prod.env.js`
npm run build

# build for staging with minification
# you should check the all environment variables in `config/staging.env.js`
npm run staging

# run linter
# npm run lint

# run unit tests
# npm run test:unit

# run e2e tests
# npm run test:e2e

# you can see more commands in `package.json` file

```
